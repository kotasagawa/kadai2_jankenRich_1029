# 課題　 じゃんけんアプリ　リッチver.

## ①課題内容（どんな作品か）
- じゃんけんで６回勝つと相手に勝利。じゃんけんに勝つごとに必殺技が相手に向かって伸びるようにしました。６回勝つと、必殺技が相手にヒットして勝利です。

## ②工夫した点・こだわった点
- 必殺技（かめはめ波）が伸びるように見せたこと。
- CSSを使って必殺技の色の調整をすることで、より本物に近づけました。
- ６回勝利することで、相手のキャラクターの画像を変えて、倒れてみえるようにしました。

## ③難しかった点・次回トライしたいこと(又は機能)
- 音声を自動で再生されるようにしたかったが、できなかった。
- 効果音をもう少し追加できればよかった。
- 必殺技をどう表現するか悩みました。今回はspanタグを追加することで必殺技を表現しました。画像のほうが簡単そうですが、画像間のつなぎ目ができそうなので、今回はやめました。

## ④質問・疑問・感想、シェアしたいこと等なんでも
- [質問]Google Chromeでは動画や音声の自動再生はできないのでしょうか？
- [疑問]必殺技を表現する部分について、今回はposition:absoluteで位置を調整しました。しかし、「キャラ、必殺技、キャラ」をdivで横並びにして表現したほうがきれいに見えるかもしれないと思いました。どのようにコーディングするのが良かったのが、疑問に思っています。
- [感想]じゃんけんアプリ　リッチverを作るにあたり、もう一つ新しいものを作ったほうが成長するかと考えましたが、新たなアイデアを出すのに時間がかかりました。しかし、前回作った経験があったので、応用して時短できた部分もあり、数をこなすことでアイデアを形にする時間も短くなるんだと感じました。
